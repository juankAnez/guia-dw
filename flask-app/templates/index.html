    📖 Acerca de esta guía

    Guía Práctica para el Parcial — Desarrollo Web & Bases de Datos II

    Esta es una sub-guía enfocada y optimizada para la preparación del examen parcial de la asignatura. Contiene únicamente los pasos esenciales para construir un CRUD completo de una sola entidad con Node.js, Express, Sequelize y Angular, eliminando la complejidad de temas como autenticación JWT, paginación y múltiples tablas que se ven en la guía completa.

    Todo el contenido aquí presentado está basado en la guía docente original del profesor, resumido y adaptado para que puedas repasar de forma rápida y práctica antes del parcial. Incluye un generador de código multi-motor (MySQL, PostgreSQL, SQL Server, Oracle) para que adaptes los ejemplos a cualquier base de datos.
